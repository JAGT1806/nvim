## installation

### Linux
``` sh 
git clone https://github.com/JAGT1806/nvim ~/.config/nvim/
```


### Windows

#### Powershell
``` sh
git clone https://github.com/JAGT1806/nvim $env:LOCALAPPDATA\nvim
```

> [!NOTE]
> LOCALAPPDATA = ~/AppData/Local/
